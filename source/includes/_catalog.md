# ODPV Value


| Term | Type | Description | Used in |
| --- | --- | --- | --- |
| `UseCase` | object | A practical business, operational, analytical, or technical scenario where one or more data products are used to create value. | ODPC, ODPG |
| `BusinessObjective` | object | A business goal or intended outcome that data products, use cases, or portfolio actions support. | ODPC, ODPG |
| `KPI` | object | A measurable indicator used to track progress toward a business objective, outcome, or portfolio goal. | ODPC, ODPG |
| `Impact` | object | The expected or observed effect created by a data product, use case, signal, or portfolio decision. | ODPC, ODPG |
| `Signal` | object | An observed demand, trend, risk, gap, usage pattern, policy change, market change, or operational event that may require action. | ODPC, ODPG |
| `Gap` | object | A missing data product, dataset, capability, quality level, access method, relationship, or portfolio coverage area. | ODPC, ODPG |
| `Priority` | classification | A ranking or importance level used to guide portfolio planning, product development, investment, or remediation. | ODPC, ODPG |
| `Outcome` | object | A desired or achieved result connected to a business objective, use case, KPI, or impact statement. | ODPC, ODPG |
| `DataNeed` | object | A required data input, data product, dataset, attribute, capability, or access pattern needed to support a use case or objective. | ODPC, ODPG |
| `Benefit` | object | A positive business, operational, financial, social, or technical value expected from a data product, use case, or portfolio action. | ODPC |
| `Demand` | object | Evidence of need or interest from consumers, stakeholders, systems, markets, policies, or operational processes. | ODPC, ODPG |
| `Opportunity` | object | A possible area for value creation, reuse, innovation, service improvement, monetization, or better decision-making. | ODPC, ODPG |
| `Risk` | object | A possible negative event, weakness, exposure, or uncertainty that may affect data product value, trust, access, compliance, or operation. | ODPC, ODPG |
| `ValueProposition` | object | A short explanation of why a data product, use case, or portfolio item matters and what value it is expected to create. | ODPS, ODPC |
| `PortfolioPriority` | classification | A portfolio-level priority assigned to a data product, use case, signal, gap, or objective. | ODPC, ODPG |
